Projeto de certificação 2 – Postagem de blog

#Rotina Invisível - Blog Minimalista

Um projeto de blog focado na apreciação da rotina cotidiana, permitindo que usuários interajam através de comentários/postagens simuladas via API.

##Tecnologias Utilizadas
- HTML5: Estrutura semântica.
- CSS3: Design responsivo com o uso de Flexbox e CSS Variables.
- JavaScript (ES6+): Manipulação de DOM e integração com API.
- Google Fonts: Utilização das fontes Lora, Montserrat e Open Sans.
- JSONPlaceholder: Simulação de requisições HTTP POST.

##Funcionalidades

Design Responsivo: Interface limpa que se adapta a diferentes tamanhos de tela.
Sistema de Postagem Dinâmico: Ao clicar em "Postar um Comentário", um formulário é revelado.
Integração com API: As postagens são enviadas para a JSONPlaceholder via fetch API, simulando o comportamento de um backend real.
Feedback em Tempo Real: O conteúdo enviado é renderizado instantaneamente na tela após a resposta da API.

##Como rodar o projeto

- Clone este repositório ou copie o código.
- Certifique-se de ter um arquivo de imagem chamado banner.png no mesmo diretório para a visualização completa.
